# Lidl Promotions Fetcher

This project fetches promotional data from Lidl Bulgaria and processes it for reporting.

## Dependencies & Linting

This project uses [uv](https://docs.astral.sh/uv/) for dependency management and linting.

### Setup
To install dependencies, run:
```bash
uv sync
```

### Running Linting/Formatting
To format the code using `black`, run:
```bash
uv run black fetch_promos.py
```
